当未来可以被模拟，我们离预知还有多远？
昨晚，我经历了一个新的「震撼时刻」，上次这种时刻还是「人生k线」的问世。
这个产品的名字是「MiroFish」(@mirofish_ai) ，仅仅只是一个小小的demo 片段，却让我盯着屏幕沉思了2个小时。

它有多么不可思议，Github中对 MiroFish 的介绍是这样的：
MiroFish 是一款基于多智能体技术的新一代 AI 预测引擎。通过提取现实世界的种子信息（如突发新闻、政策草案、金融信号），自动构建出高保真的平行数字世界。在此空间内，成千上万个具备独立人格、长期记忆与行为逻辑的智能体进行自由交互与社会演化。
简单来说，你上传材料，MiroFish会根据材料生成多个模拟现实人类的Agent，让它们相互作用，推演未来。想象一下，假如未来可以被AI模拟，我们离预知还有多远。
思考一：Mirofish改变的是人们的决策方式
以前，我们一般是怎么决策的：
层级一：经验与直觉。通过过去经历形成的认知，做出判断。
层级二：专家咨询。通过咨询专家或者咨询公司，辅助做出决策。（甚至玄学咨询）
层级三：数据分析。运用软件分析海量数据做出判断。
现在：通过AI (MiroFish) 模拟未来，做出判断。

想象一下：
如果西贝用 MiroFish 进行公关指导，贾国龙可能就不会宣布西贝大规模闭店。

如果锤子手机用 MiroFish 预测市场反应，罗永浩现在可能就不会在做播客了。
我认为 MiroFish 最伟大的创新点之一在于其一定会改变未来世界的决策模式，小到个人每日决策，大到国家政策，谁不想用一个沙盘进行模拟辅助判断。
思考二：Agent究极进化论：从回答过去到预演未来
AI agent 1.0：单兵作战。你给agent一个待解决的问题，agent独立思考、调用工具、完成任务。
AI agent 2.0：Agent Teams。相比于1.0时代， agent 学会了分工协作，学会了更高效的完成任务。

需要补充的是，Agent 1.0 和 Agent 2.0 时代有一个共识的问题，模型都会受困于过去的数据。它就像一个学生反复做往年的真题，也许能越做越快、越做越准，但它的解题思路永远来自历史的经验。当面对真正前所未有的、没有历史参照的全新挑战时，这套方法就触及了它的极限。
Agent 3.0 时代：以 MiroFish 为代表的群体智能，是一个由成千上万个具有独立人格和长期记忆的Agents构成的「数字社会」。这些Agent在一个类似于真实世界的平行世界中自由交互、碰撞、演化，并推理出未来的结果。当成千上万个Agent产生真实的社会交互时，它们本身就在创造新的、不存在于任何历史数据中的信息。
为了方便大家理解我的观点，我用作者发布的红楼梦Demo来举例。这个Demo导入了《红楼梦》原著前80回的故事，让AI自行推演后续剧情，再与现有的后40回进行对比。

MiroFish从前80回的故事中提取出贾宝玉、林黛玉等角色，生成一群具有独立人格的Agent。这些Agent在数字世界中按照各自的性格自由交互，推演出属于它们自己的结局。而这些结局，是不存在于任何历史数据中的全新内容。
具体的预测结果建议大家直接观看作者发布的demo视频，看完你会和我一样震撼。（链接在最后）
思考三：超级个体时代已经到来
MiroFish 作者是大四的学生，项目是10天vibe coding出来的，并获得了盛大3000万的投资。

他当前vibe coding的工作流是：
figma 设计草图 -  google ai studio 快速复制一个前端demo - 把页面补进项目文档，再把任务拆成模块，交给AI IDE 分批开发。

与此同时，他也在文章分享了些经验，包括可以使用多agent 并行一个任务等等。（以上总结自作者小红书文章）
通过阅读作者整篇文章，我最大的感受是超级个体时代已经来临，因为市场正在疯狂寻找那些能把AI转化为生产力的人，并且这样的案例正在一个个出现。在这个时代，真正让一个人脱颖而出的，是敢于创造的精神、独特的审美和丰富的想象力。毕竟，AI 让一切皆有可能。

最后借用Mirofish创始人的一句话：
Vibe coding 做产品、做开源，是一条切实可行、可复制的路径，而且现在还不拥挤。
AI时代已经到来！至于如何play this game, it depends on you!
